# Sweets Documents

## Tpay API Documents ＆ Moc Server
aglioとdrakovを使ってAPI DocumentとMoc Serverを立ち上げられます。

### Requirement
- Docker
- Docker Compose

### Quick Start
```shell
$ docker-compose build
$ docker-compose up -d

# Access to API Document: http://0.0.0.0:4000/
# Access to Moc Server: http://0.0.0.0:4001/
```
